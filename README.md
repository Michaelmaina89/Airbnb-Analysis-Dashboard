# Global Airbnb Market Analysis - Power BI

### Project Overview
An end-to-end data analysis project exploring 265k listings and 5.3M reviews across 10 global cities.

### Key Features
* **Outlier Management:** Filtered extreme price values (up to $625k) to normalize data.
* **AI Insights:** Used Key Influencers to identify that Room Type and Capacity drive $90+ price jumps.
* **Trend Analysis:** Visualized the 2014-2019 growth boom and the 2020 COVID-19 recovery.

### Tools Used
* Power BI Desktop
* Power Query (ETL)
* DAX (Custom Measures)
* Git LFS (Large File Management)
